# Data Jobs Dashboard – Power BI Project

## Overview
The **Data Jobs Dashboard** is an interactive Power BI project designed to analyze job market trends, salary insights, and employment patterns in the data industry.

This dashboard helps users explore job postings, salaries, remote work opportunities, hiring trends, and job benefits using interactive visualizations and drill-through functionality.

---


![Main Dashboard](/images/main-dashboard.png)

---

## Features

- Interactive Power BI Dashboard
- Drill-through functionality for detailed analysis
- Dynamic filtering and slicing
- Salary analysis and job trend visualization
- Geographic analysis of job postings
- KPI cards for quick insights
- Responsive and user-friendly dashboard design
